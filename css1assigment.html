CSS can be added to HTML document in 3 way:

  • inline: by using the style attibute inside the html elements.
  • internal: by using a style element in head section.
  • External; by using the link element to link to an external CSS file.
  • the most common way to add css ,is to keep the styles in external css files. However, int this toturail we will use inline and internal styles ,because this is easier to demonstrate, and easier for you to try it yourself.

when comparing css class vs id ,the difference id that css class applies a style to multiple element .ID, on the other hand ,applies a style to one unique element. ID is also specialin that you can use a special URL to link directly to an element and it's used by java script

In css, selector are used to target a specific element or range of element on a webpage. Once an element has been targeted , a style or set of styles can be applied to the element.

There is a wide range of selectors avaailable.Two of the most commonly used are class and ID. Both are used totaget elements to which a style should be applied.
